# 🎬 AI Movie Recommendation System (Tkinter)

## 📌 Project Overview
This is a simple desktop-based **Movie Recommendation System** built using Python and Tkinter GUI.  
The system recommends movies based on the user's selected genre.

---

## ⚙️ Features
- Easy-to-use GUI (Tkinter)
- Dropdown menu for genre selection
- Instant movie recommendations
- Clean and simple interface
- Works offline (no internet required)

---

## 🎯 Supported Genres
- Action
- Comedy
- Horror
- Science Fiction

---

## 🧠 How It Works
1. User selects a movie genre from dropdown.
2. Clicks on **"Get Recommendations"** button.
3. System displays a list of recommended movies for that genre.

---

## 🛠️ Technologies Used
- Python
- Tkinter (GUI Library)

---

## 🚀 How to Run
1. Install Python (if not installed)
2. Save the file as project3.py
3. Run the file using command:

```bash
python movie_recommendation_system.py
```

---

## 📷 Output Preview
- A window opens with a dropdown menu
- After selection, recommended movies appear in text box
