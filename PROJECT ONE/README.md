# 🤖 Smart GUI Chatbot

A colorful and interactive GUI-based chatbot made using Python and Tkinter.

## ✨ Features

* Modern colorful interface 🎨
* Smart chatbot replies 🤖
* Time and Date support ⏰
* Random jokes 😂
* Motivational quotes 🌟
* AI and Python information 🐍
* User-friendly GUI 🖥️

---

## 🛠️ Technologies Used

* Python
* Tkinter

---

## ▶️ How to Run

### Step 1: Install Python

Make sure Python is installed on your computer.

### Step 2: Save the File

Save the chatbot code as:

```bash
gui_chatbot.py
```

### Step 3: Run the Project

Open terminal in VS Code and run:

```bash
python gui_chatbot.py
```
